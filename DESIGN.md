# Design System

## Direction
White gallery surface, dark cinematic hero, smoky gray about band, restrained burnt-orange icon accent. The page should preserve the original Tilda structure while improving image rhythm, mobile behavior, and polish.

## Palette
- Background: `oklch(0.985 0.004 80)`
- Text: `oklch(0.13 0.006 65)`
- Muted text: `oklch(0.42 0.006 65)`
- Soft border: `oklch(0.86 0.006 70)`
- About band: `oklch(0.72 0.006 72)`
- Accent: `oklch(0.66 0.17 43)`
- Dark hero: `oklch(0.12 0.008 70)`

## Typography
- Display: `Oswald`, fallback `Arial Narrow`, `Arial`, sans-serif.
- Body and UI: `Manrope`, fallback `Arial`, sans-serif.
- Hero and section headings use strong weight and tall condensed rhythm.
- Body copy stays under 75ch and keeps generous line height.

## Components
- Header: white, minimal text navigation, circular outline social buttons.
- Hero: full-bleed image with dark edge treatment and centered type.
- Gallery: original Tilda image set only, using controlled triptychs, two-column collages, and two clean three-column rows instead of freeform masonry.
- About: image plus copy in a gray band.
- Contact: large heading and three circular social actions.
- Lightbox: full-screen image viewer with keyboard and click-close behavior.
